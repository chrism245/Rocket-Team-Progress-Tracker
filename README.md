# chrism245 — Browser Scripts & Extensions

A collection of Tampermonkey userscripts and browser extensions for Zendesk and WordPress/WooCommerce workflows.

---

## 🚀 [Rocket Team Progress Tracker](./Rocket-Team-Progress-Tracker/)

A Tampermonkey userscript for Zendesk agents. Automatically tracks daily ticket interactions (private notes) and displays them in a visual calendar with product badges, daily progress counter, and milestone animations.

**Version:** 1.3.6 · **Type:** Tampermonkey userscript · **Platform:** Zendesk

---

## 🔍 [WooCommerce Forum Thread Manager](./WooCommerce-Forum-Thread-Manager/)

Userscript for managing open/unresolved WooCommerce forum threads.

---

## 🕷️ [WooCommerce Forum Scraper](./WooCommerce-Forum-Scraper%202.3.3/)

Chrome extension for scraping and reviewing WooCommerce forum threads.
