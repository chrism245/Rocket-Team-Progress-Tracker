# chrism245 — Browser Scripts & Extensions

A collection of Tampermonkey userscripts and browser extensions for Zendesk workflows.

---

## 🚀 [Rocket Team Progress Tracker](./Rocket-Team-Progress-Tracker/)

A Tampermonkey userscript for Zendesk agents. Automatically tracks daily ticket interactions (private notes) and displays them in a visual calendar with product badges, daily progress counter, and milestone animations.

**Version:** 1.3.6 · **Type:** Tampermonkey userscript · **Platform:** Zendesk
